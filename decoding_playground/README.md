# Neural Text Decoding Playground

An interactive, fully transparent playground for neural text decoding strategies that reproduces the core ideas of **"The Curious Case of Neural Text Degeneration" (Holtzman et al., 2020)**. It runs GPT-2 (124 M) entirely on CPU — no GPU or internet connection required after the first model download. To get started, install the dependencies with `pip install -r requirements.txt` (Python 3.10+ recommended), then launch the app with `streamlit run app.py`; a browser tab will open automatically at `http://localhost:8501`. Type a prompt, choose one of the six decoding strategies (Greedy, Beam Search, Pure Sampling, Temperature, Top-k, or Nucleus/Top-p), and click **Step Once** to generate one token at a time — each step shows a colour-coded bar chart of the top-15 candidate tokens (green = kept, red = cut, gold = sampled), a PEAKED/FLAT distribution badge explaining model confidence, and a plain-English sentence describing why that token was chosen. Use **Generate N** followed by the replay slider to scrub back through any step in a full generation run, or **Reset** to start over. To verify the backend independently (no browser required), run `python decoder.py`.
